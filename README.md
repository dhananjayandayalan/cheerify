# Cheerify

A web application that generates **truly unique**, AI-powered personalized cheer-up notes for Twitter users by analyzing their Twitter personality and creating custom PDF messages.

## Features

- **AI-Powered Unique Notes**: Every note is 100% unique, generated by Google Gemini AI based on real personality analysis
- **FREE AI Usage**: Uses Gemini 1.5 Flash with generous free tier (15 requests/minute, 1 million tokens/day)
- **No Default Templates**: Never shows generic messages - either personalized or error
- **Web Scraping**: Scrapes public Twitter profiles (no costly API needed)
- **Beautiful PDFs**: Creates professionally formatted PDF notes with custom layout
- **Dark/Light Mode**: Toggle between light and dark themes
- **Typewriter Effect**: Animated text display for a delightful user experience
- **Smart Error Handling**: User-friendly error popups when analysis fails

## How It Works

1. Enter a Twitter handle
2. The app scrapes the user's public profile, tweets, and bio using Puppeteer
3. Analyzes their personality traits (technical, creative, helpful, humorous, tone style, topics)
4. Sends the analysis to Google Gemini 1.5 Flash to generate a completely unique personalized note
5. Displays the note with a beautiful typewriter effect
6. Download as a professionally formatted PDF

## Project Setup

### Prerequisites

- Node.js (v20.19.0 or v22.12.0+)
- npm

### Installation

```sh
npm install
```

### Google Gemini API Configuration (Required)

To generate unique AI-powered notes:

1. **Get Gemini API Key** (FREE):
   - Go to [Google AI Studio](https://aistudio.google.com/app/apikey)
   - Click "Create API Key"
   - Copy the generated key

2. **Configure Environment Variables**:
   - Copy the `.env.example` file to `.env`:
     ```sh
     cp .env.example .env
     ```
   - Open `.env` and add your Gemini API key:
     ```
     GEMINI_API_KEY=your_actual_api_key_here
     ```

3. **Cost Information**:
   - Uses Gemini 1.5 Flash model
   - **FREE TIER**: 15 requests per minute, 1 million tokens per day, 1500 requests per day
   - Each note generation is FREE within these limits
   - Much better than paid APIs for this use case!

**Note**: The app uses a backend server to handle scraping and AI generation. The server runs on port 3001 by default.

### Development

Start both frontend and backend servers:

```sh
npm run dev
```

This will start:
- Frontend (Vite): `http://localhost:5173`
- Backend (Express): `http://localhost:3001`

Or run them separately:
```sh
npm run dev:client  # Frontend only
npm run dev:server  # Backend only
```

### Production Build

Type-Check, Compile and Minify for Production:

```sh
npm run build
```

## Type Support for `.vue` Imports in TS

TypeScript cannot handle type information for `.vue` imports by default, so we replace the `tsc` CLI with `vue-tsc` for type checking. In editors, we need [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar) to make the TypeScript language service aware of `.vue` types.
